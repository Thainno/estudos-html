  3. Escovar os dentes
  4. Ir para o trabalho

Caso necessário inserir uma atividade antes de outra, basta inserir com a tag 'li' que a numeração já fica correta.

Reprodução de áudios

Utilizamos a tag (audio)

Alguns navegadores nãos suportam o formato mp3, podemos utilizar: WAV ou OGG.

Reprodução de vídeos

Consome dados do seu servidor de hospedagem.

Temos os formatos mp4, webm e ogv

Baixar 'Handbrek' para deixar o vídeo mais leve e menor.

Alguns navegadores nãos suportam o formato mp4, podemos utilizar: m4v e webm.

Inserindo vídeo do YouTube

Não consome dados do seu servidor, consome dados do YouTube.
Compensa criar um canal no YouTube e colocar o vídeo em seu site.
