Favicon

É o ícone da página presente ao lado do title.

É recomendado utilizar imagens com a extenção '.ico'
As página indicadas para criar o seu favicon:
Imagens pré-definidas: https://iconarchive.com/
Criar com pixel art: https://www.favicon.cc/
Para criar a partir de imagem, texto ou emojin: https://favicon.io/

A imagem que utilizamos foi:

Basta inserir a tag <link:"favicon"> no cabeçalho <head>.

Imagens dinâmicas

Tente abrir esse site em vários dispositivos diferentes, ou simplesmente aumente ou diminua o tamanho de seu navegador para testar essa funcionalidade.

Utilizamos a tag <picture> e <source media type>.
OBS.: é necessário inserir as imagens em ordem obedecendo os tamanhos.

Foi utilizado 3 imagens de tamanhos diferentes para essa funcionalidade, sendo elas alteradas com o tamanho de página em 750px (pequena para a média), e 1050px (média para a grande) respectivamente.
