Como inserir símbolos com HTML5

Para que o símbolo seja carregado na maioria dos navegadores, é recomendado inserir da seguinte forma:


Símbolo: ™
Código : &trade
Símbolo: ∑
Código : &sum
Símbolo: ←
Código : &larr
Símbolo: →
Código : &rarr HTML Trademark Symbol HTML Trademark Symbol HTML Trademark Symbol HTML Trademark Symbol
