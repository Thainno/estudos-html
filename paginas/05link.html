Acesso a outras páginas de seu site

Para inserir Hyperlinks, basta inserir a tag <a href="link aqui">.
Use a tag: <rel="next"> para inserir uma referência que está passando uma página e a tag: <rel="prev"> para inserir uma referência que está retornando uma página.

Acesso a outros sites

Para inserir Hyperlinks, basta inserir a tag <a href="link aqui">.
Use o <target="_blank" rel="external"> para links externos que não fazem parte do seu site, assim abre uma nova janela e não sai do seu site.

Exemplos

Você pode acessar o meu github pelo link: GitHub
Você também pode acessar o meu LinkedIn pelo link: LinkedIn

Mais atributos para referência do link pode ser encontrados em sites como: w3schools.com

Links para Download

Para inserir Hyperlinks de download, basta usar a tag:
<a href="link do arquivo aqui" download="nome do arquivo aqui" type="application">

  • Clique aqui para baixar o arquivo teste pdf.
  • Clique aqui para baixar o arquivo teste word.

Você pode encontrar todas as referências para download de arquivos em: https://iana.org
