Listas ordenadas

É criada com a tag <ol> e ramificada com a tag <li>.

  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Ir para o trabalho

Caso necessário inserir uma atividade antes de outra, basta inserir com a tag <li> que a numeração já fica correta.
OBS.1: Não precisa fechar a tag após a versão 5 do HTML.
Com o parâmetro type podemos alterar o tipo de lista.
Como o parâmetro start podemos indicar em qual valor a lista começa (deve ser sempre numérico).

Listas não ordenadas

É criada com a tag <ul> e ramificada com a tag <li>.

  • Tomate
  • Batata
  • Cenoura
  • Feijão

Podemos atribuir os <type>: 'square', 'circle' ou 'disc'.

Juntando Listas - atividades diárias

Junção das tags <ol> e <li>.

  1. Obrigatórias
    1. Trabalhar
    2. Comer
    3. Dormir
  2. Opcionais
    1. Estudar
    2. Jogar
    3. Entretenimento

Juntando Listas - atividades semanais

Junção das tags <ul>, <ol> e <ol>.

  • Segunda
    1. Comer
    2. Dormir
  • Terça
    1. Dormir
    2. Trabalhar
  • Quarta
    1. Trabalhar
    2. Comer
